# Portfolio

GitHub Pages에서 포트폴리오 PDF를 미리보고 다운로드할 수 있는 정적 페이지입니다.

## GitHub Pages 설정

1. Repository Settings로 이동합니다.
2. Pages 메뉴에서 Build and deployment의 Source를 `Deploy from a branch`로 선택합니다.
3. Branch를 `main`, 폴더를 `/ (root)`로 선택한 뒤 저장합니다.

배포가 완료되면 아래 주소에서 확인할 수 있습니다.

https://ch-hyuk.github.io/Portfolio/
